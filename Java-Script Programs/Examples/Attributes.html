Changing Element Attributes

  1. JavaScript allows you to modify an element’s attributes dynamically.
    • setAttribute(attribute, value) → Updates an attribute.
    • getAttribute(attribute) → Retrieves an attribute.
    • removeAttribute(attribute) → Removes an attribute.
